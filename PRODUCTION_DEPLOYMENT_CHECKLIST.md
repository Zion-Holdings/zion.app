```bash
# Check Node.js installation

# Check dependencies
npm install

# Check port availability

```

#### **Connection Issues**
```bash
# Check network connectivity

# Check firewall settings
sudo ufw status

# Restart discovery

```

#### **Performance Issues**
```bash

# Check resource usage
```
