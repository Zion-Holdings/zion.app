## Best Practices

- **Consistent Keys**: Use descriptive, hierarchical keys (e.g., `nav.home`, `errors.page_not_found`)
- **Pluralization**: Use i18next pluralization features for count-based translations
- **Context**: Provide context in key names to help translators
- **Interpolation**: Use variables for dynamic content: `"welcome": "Hello, {{name}}!"`
- **Testing**: Test all languages in different browsers and devices
