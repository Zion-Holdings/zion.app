# API Documentation

## Auth
Endpoints for user registration, login, and token refresh. OAuth is supported for third-party connections.

## Jobs / Talent
APIs to list jobs, create proposals, and manage talent profiles.

## Quotes / Milestones
Tools for creating proposals, setting milestone payouts, and tracking progress.

## Messaging
Real-time chat and notifications between clients and talent.

## Voting
Submit and tally on-chain votes via API gateways.

## Token Balances / Transfers
Query wallet balances and initiate transfers of ZION$ tokens.
