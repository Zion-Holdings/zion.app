# 🤖 Zion App - Autonomous Automation System

A comprehensive, AI-powered autonomous automation system that runs independently and continuously improves your application using multiple AI agents and modern automation patterns.

## 🚀 Overview

The Autonomous Automation System is a next-generation automation platform that operates completely independently with AI-powered decision making, self-healing capabilities, and continuous improvement. It consists of multiple specialized agents that work together to monitor, optimize, and enhance your application 24/7.

## 🚀 Quick Start

### 1. **Installation**

```bash
# Clone the repository
git clone <repository-url>
cd bolt.new.zion.app

# Install dependencies
npm install

# Setup environment
cp .env.automation.template .env.automation
# Edit .env.automation with your API keys
```

### 2. **Configuration**

Create a `.env.automation` file with your API keys:

```env
# AI Provider API Keys
CURSOR_API_KEY=your_cursor_api_key
OPENAI_API_KEY=your_openai_api_key
CLAUDE_API_KEY=your_claude_api_key

# System Configuration
AUTOMATION_PORT=3001
NODE_ENV=production

# Optional: Local AI Configuration
LOCAL_AI_ENABLED=true
LOCAL_AI_ENDPOINT=http://localhost:11434
LOCAL_AI_MODEL=codellama:7b
```

### 3. **Start the System**

```bash
# Start all autonomous agents
./start-autonomous-automation.sh start

# Or start individual agents
node scripts/autonomous-automation-system.js
node scripts/ai-code-review-automation.cjs
node scripts/performance-optimization-automation.cjs
node scripts/security-monitoring-automation.cjs
```

### 4. **Access the Dashboard**

Open your browser and navigate to:

```
http://localhost:3000/autonomous-automation-dashboard.html
```

```javascript
  // 1. Analyze the error
```

## 🔧 Troubleshooting

### **Common Issues**

1. **Agent Not Starting**

   ```bash
   # Check if port is available
   lsof -i :3001

   # Check logs
   tail -f logs/autonomous-automation.log

   # Restart agent
   ./start-autonomous-automation.sh restart
   ```

2. **AI Providers Not Working**

   ```bash
   # Check API keys
   echo $CURSOR_API_KEY
   echo $OPENAI_API_KEY

   # Test API connection
   curl -H "Authorization: Bearer $CURSOR_API_KEY" \
        https://api.cursor.sh/v1/workspaces
   ```

3. **Performance Issues**

   ```bash
   # Check system resources
   top
   htop

   # Check agent resource usage
   ps aux | grep node

   # Restart with resource limits
   NODE_OPTIONS="--max-old-space-size=2048" node scripts/autonomous-automation-system.js
   ```

