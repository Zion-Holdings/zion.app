# Authentication API (AUTH_API.md)

This module handles user authentication and session management within the Zion OS ecosystem. It provides methods for logging in with various credentials, logging out, and retrieving user information.

## Methods

_Detailed function definitions, parameters, and return types will be specified here._

**Examples:**

*   `loginWithEmail(email, password)`
*   `loginWithWallet(walletProvider, signature)`
*   `logout()`
*   `getUser()`
*   `refreshToken()`
*   `ssoLogin(provider)`
